# X (Twitter) Style Guide

## Character Limits
- **Free users**: 280 characters per tweet.
- **Premium users**: up to 25,000 characters (but long posts show a truncated 280-char preview with "Show more").
- **Default to 280 characters** unless the user specifies they have Premium.

## Tone
- Punchy, conversational, opinionated.
- Write like you're texting a smart friend, not writing a report.
- Hot takes and strong opinions perform well — lukewarm statements get ignored.

## Format
- For 280-char tweets: one clear idea per tweet. No filler words.
- For longer content: consider a thread (multiple connected tweets). Number them (1/5, 2/5...) or use a hook tweet + replies.
- Thread structure: hook tweet first (the one people see in feed), then supporting points.
- Emojis are fine in moderation — don't overdo it.

## Do
- Start with a hook or surprising statement.
- Use plain language. Remove every word that doesn't earn its place.
- Quote-tweet or reply to relevant conversations when the content fits.
- Include a link in the last tweet of a thread, not the first (algorithm penalty).

## Don't
- Don't write like a LinkedIn post (too formal, too long).
- Don't use hashtags excessively — 0–1 is fine, 5+ looks spammy.
- Don't start with "I'm excited to announce..." — nobody cares, get to the point.
- Don't thread things that can be said in one tweet.
